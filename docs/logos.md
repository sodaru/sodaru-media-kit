# Logos

---

## Usage

- **Use A Logo**

  ```typescript
  import { SodaruLogo } from "@solib/media-kit";
  ```

  Works as a Normal React Component  
  _Props_

  - all props of `SVGSVGElement`
  - `sx` of Material UI

- **Use Logos Dashboard**

  ```typescript
  import { Logos } from "@solib/media-kit";
  ```

  Displays all available logos and button to download .svg and .png
